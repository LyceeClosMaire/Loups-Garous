Simple variable et filtres

La valeur d'une variable du template peut être insérée simplement : {{ machin }} .

Il est également possible de passer cette valeur à travers une moulinette appelée un "filtre" dont la liste peut être trouvée dans la doc : {{ machin|upper }} ou {{ machin|capitalize }} ou {{ machin|replace("uc", "ash")|capitalize }}...
